## ✨ Premium Experience Section

Premium TV Player is designed as a **high-end living room product**, not as a basic stream list app.

### Premium pillars
- **Cinematic presentation:** Large hero areas, curated spacing, premium typography, and elegant transitions.
- **TV-native interaction:** Built for remote navigation, focus clarity, and comfortable 10-foot readability.
- **Product-grade architecture:** Backend-driven entitlement and device model with cloud sync readiness.
- **Consistent design language:** Reusable design tokens and scalable UX patterns across platforms.
- **Long-term platform vision:** Android TV first, then mobile, admin web, and future TV ecosystems.

### Premium UX goals
- Zero clutter, strong hierarchy, high visual contrast.
- Fast, predictable navigation with clean focus states.
- Emotionally “premium” onboarding and home browsing flow.
- Stable account experience across devices and profiles.

Premium TV Player is a private premium TV player project built to deliver a polished, cinematic, and account-centric viewing experience.

### What this project is
- A premium application framework for TV playback experiences.

## 🏗 Architecture Snapshot

### Client Stack
- Kotlin + Jetpack Compose + Compose for TV
- Media3 / ExoPlayer
- Hilt, Coroutines, Flow
- Navigation-Compose
- Room + DataStore

### Backend Stack
- NestJS (TypeScript)
- PostgreSQL + Prisma
- Redis
- Firebase Admin
- REST API contracts

### Shared Foundation
- API contracts
- Domain models
- Parsers
- i18n keys
- UI tokens
- Entitlement engine
